# FluxChatService
FluxChatService is a Custom Chat system for roblox which the original was ExpChatRemake. This includes many features.

# Notes
- You can keep the Package object if you want the chat to recieve updates.
- You must disable CreateDefaultTextChannels in TextChatService

# Setup
- Drag the 'FluxChatService' Folder into 'ServerScriptService'
- Go to Security Settings in File/Experience Settings, and turn on 'Enable Studio Access to API Services'
- Then your done

Enjoy the chat!
